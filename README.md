# Risk Game

A simple digital implementation of the board game **Risk**, built in Java for our Software Quality Engineering project.

## Project Overview

This project recreates the core gameplay of Risk, including:

- Starting a new game
- Assigning territories and armies
- Reinforcement phase
- Attack phase
- Fortify phase
- Player elimination
- Win condition detection

## Tech Stack

- **Java**
- **JavaFX** for GUI
- **[Placeholder]** for testing
- **GitHub Actions** for CI
